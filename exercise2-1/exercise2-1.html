Your sketch:

Instructions

In this exercise, you will layer two images of the same size on the canvas and create a fade effect by manipulating the pixels in the top image to gradually reveal the second image. You can see an example in the video below.

You can use the provided images, night-sky.jpg and blue-sky.jpg, or you can use your own images. If you choose your own images, make sure they are the same size as each other. Make the canvas the same size as the images.

Choose one of your images to be the starting image and one to be the ending image--your fade will transition from the starting image to the ending image. Draw both images one on top of the other in draw(). The starting image should be the second image drawn.

Remember that you will need to call loadPixels() on the image before changing the values of any pixels, and that you will need to call updatePixels() on the image for your changes to take effect e.g.:

img.loadPixels();

img.pixels[index] = newValue;

img.updatePixels();

When you are finished, use check-exercise2-1.html to check your work.

Expected output:

The alpha value of each pixel in an image will be 255--fully opaque. You can create the fade effect by gradually reducing the value of the alpha value of each pixel. Remember that each pixel in an image translates to four elements in the image's pixel array. These elements correspond to the red, green, blue, and alpha values of the pixel. Therefore, to reduce the alpha values of each pixel in the image, iterate over every fourth element in the image's pixel array.
